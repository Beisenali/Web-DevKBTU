# Lab 4: Angular

Web Development Lab 4 - Angular framework.

## Structure

- **task1/** — Completed Angular "Learn Angular" tutorial from https://angular.dev/tutorials/learn-angular
- **task2/online-store/** — Online Store application with kaspi.kz product catalog

## Task 1

Angular "First App" / "Learn Angular" tutorial — housing locations app.

```bash
cd task1 && npm install && ng serve
```

Optional API: `npm run backend` (json-server) then `ng serve`.

## Task 2

The Online Store displays a catalog of products from kaspi.kz with:
- Product cards with image, name, description, price, rating
- Share to WhatsApp and Telegram
- Responsive grid layout
- Optional image gallery

Run: `cd task2/online-store && npm install && ng serve`
