Album Browser - Lab 6
=====================

This folder contains the Angular Album Browser application for Lab 6 (Routing & HTTP).

## Running the project

1. Navigate into the project:
   ```bash
   cd Lab6/album-browser
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

4. Open the app in your browser at `http://localhost:4200/`.
