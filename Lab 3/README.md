## Task 2: To-Do List Application

A simple To-Do List app in `todo-app/`.

### Features
- Add new items via text input and button
- Mark items as done with checkbox (strikethrough)
- Delete items with delete button

### Tech
- Semantic HTML: `<header>`, `<main>`, `<form>`, `<label>`, `<ul>`, `<li>`, `<button>`
- CSS: Flexbox layout, clean styling
- Vanilla JavaScript: DOM API, addEventListener, no frameworks

### Run
Open `todo-app/index.html` in a browser.
